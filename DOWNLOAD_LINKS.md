## What's Included

Both Excel files contain all the requested columns:
- ✅ Failed_Filter_Count
- ✅ All_Filters_Pass
- ✅ Any_Filter_Fail
- ✅ Status_Matches_AllFilters
- ✅ Failed_Filter_Count_17
- ✅ All_17_Filters_Pass
- ✅ Any_17_Filter_Fail

## Usage

1. Download the Excel file using the link above
2. Open in Excel or Google Sheets
3. Use as a template for your own data
4. Or use the `add_filter_columns.py` script to add these columns to your existing data

## Alternative: Generate from Your Data

If you have your own data file, you can use the script to generate an Excel file with these columns:

```bash
python3 add_filter_columns.py your_data.csv your_output.xlsx
```
